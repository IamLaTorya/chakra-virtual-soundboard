## Welcome to The Virtual Chakra Soundboard
A magical, mobile‑responsive soundboard that plays chakra‑aligned tones with 1‑minute looping, soft fade‑outs, toggle mode, countdown timers, and glowing visual feedback. To support meditation, grounding, and ritual practice.

---

## Features
- **1‑Minute Looping — each chakra tone loops automatically for 60 seconds**
- **Toggle Mode — click once to start the loop, click again to stop**
- **Soft Fade‑Out — audio fades out smoothly instead of stopping abruptly**
- **Countdown Timer — shows seconds remaining for each active sound**
- **Chakra Glow Effects — each key glows in its chakra color when activated**
- **Mobile Responsive — grid layout adapts beautifully to small screens**
- **Fixed Footer — stays anchored without overlapping the soundboard**
- **Accessible UI — large tap targets, readable text, semantic structure**
